使用说明 - WARP 本地注册工具
=========================================

这是一个 Python GUI 工具，用于在本地 Windows 环境下解决 Cloudflare WARP 注册失败的问题。

由于 VPS 的 IP 可能被 Cloudflare 封控或限制 (导致 500/403 错误)，您可以使用此工具在本地电脑（配合代理软件）生成干净的 WireGuard 配置文件，然后粘贴到 VPS 上使用。

前提条件
---------------------
1. 已安装 Python 3.x
2. 已安装 requests 库 (如果没有，请运行下方安装命令)

使用步骤
---------------------
1. 安装依赖库 (如果尚未安装):
   打开命令提示符 (cmd) 或 PowerShell，运行：
   pip install requests

2. 运行脚本:
   在当前目录下运行：
   python warp_reg_gui.py

3. 操作:
   - 界面启动后，如果您使用代理软件 (如 V2RayN/Clash)，请在顶部的 "HTTP 代理" 栏填入代理地址，例如: http://127.0.0.1:7890
   - 点击 "一键注册并生成配置" 按钮。
   - 脚本会自动下载 wgcf.exe 并进行注册。
   - 生成成功后，下方的文本框会显示 WireGuard 配置内容。

4. 應用到 VPS:
   - 点击 "复制配置内容"。
   - 回到 VPS 的安装脚本界面 (VPS-Play)。
   - 选择 WireProxy 安装模式 -> "3. 导入本地配置文件 / 手动输入" -> "手动输入"。
   - 粘贴您复制的内容，最后按回车完成。

注意
---------------------
- 如果下载 wgcf.exe 失败，您可以手动下载 wgcf (Windows amd64 版本)，重命名为 wgcf.exe 放在本脚本同级目录下。
- 如果注册提示 429 错误，请尝试更换代理节点。
